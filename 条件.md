# 条件控制 (Conditional Control)

条件语句用于根据不同的情况执行不同的代码块。

## 1. if 语句

`if` 语句是最基本的条件控制结构。

### 1.1 基本语法

```korelin
if (条件表达式) {
    // 条件为 true 时执行
} else if (条件表达式2) {
    // 条件2 为 true 时执行
} else {
    // 所有条件都不满足时执行
}
```

### 1.2 示例

```korelin
var score = 85;

if (score >= 90) {
    os.println("优秀");
} else if (score >= 60) {
    os.println("及格");
} else {
    os.println("不及格");
}
```

## 2. switch 语句

`switch` 语句用于多路分支选择，通常比多个 `if-else` 更清晰。

### 2.1 语法

```korelin
switch (表达式) {
    case 值1:
        // 执行代码
        break; // 必须 break，否则会穿透
    case 值2:
        // 执行代码
        break;
    default:
        // 默认执行代码
}
```

### 2.2 示例

```korelin
var int day = 3;

switch (day) {
    case 1:
        os.println("Monday");
        break;
    case 2:
        os.println("Tuesday");
        break;
    case 3:
        os.println("Wednesday");
        break;
    default:
        os.println("Other day");
}
```

> **注意**：
> 1.  目前 `switch` 主要支持整数和字符类型的匹配。
> 2.  每个 `case` 块结束后通常需要 `break` 语句来跳出 `switch`。

## 3. 三元运算符 (Ternary Operator)

Korelin 支持 C 风格的三元运算符 `? :`，用于简单的条件赋值。

```korelin
// 语法: 条件 ? 真值 : 假值
var max = (a > b) ? a : b;
```
