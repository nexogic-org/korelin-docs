Korelin 语言文档

欢迎来到 Korelin 语言官方文档。Korelin 是一门运行在 KVM 上的静态类型、面向对象编程语言。

最新更新： 嵌入式引擎已重命名为 kembed。请参考 C/C++ API 指南 了解详情。

快速开始

// main.kri
import os;

void main() {
    os.println("Hello, Korelin!");
}

运行：korelin run main.kri
