# 循环 (Loops)

## 概述
循环语句用于重复执行一段代码。Korelin 支持 `for`、`while` 和 `do-while` 三种循环结构。

## For 循环

`for` 循环是最常用的循环结构，通常用于已知循环次数或需要初始化、条件检查和步进操作的场景。

### 语法定义
```korelin
for (initialization; condition; increment) {
    // 循环体
}
```

*   **initialization**: 初始化语句，在循环开始前执行一次。通常用于声明或初始化循环变量。
*   **condition**: 循环条件表达式，在每次迭代前求值。如果为 `true`，则执行循环体；如果为 `false`，则终止循环。
*   **increment**: 步进语句，在每次循环体执行完毕后执行。

### 示例

```korelin
// 标准 for 循环
for (var i = 0; i < 10; i = i + 1) {
    print(i);
}

// 无限循环
for (;;) {
    // 需要内部 break 跳出
}
```

## While 循环

`while` 循环在每次迭代前检查条件。

### 语法定义
```korelin
while (condition) {
    // 循环体
}
```

### 示例
```korelin
var i = 0;
while (i < 5) {
    print(i);
    i = i + 1;
}
```

## Do-While 循环

`do-while` 循环先执行一次循环体，然后检查条件。保证循环体至少执行一次。

### 语法定义
```korelin
do {
    // 循环体
} while (condition);
```

## 循环控制

*   `break`: 立即终止当前循环。
*   `continue`: 跳过当前迭代的剩余部分，进入下一次迭代。

```korelin
for (var i = 0; i < 10; i = i + 1) {
    if (i == 5) {
        break; // 当 i 为 5 时退出循环
    }
    if (i % 2 == 0) {
        continue; // 跳过偶数
    }
    print(i);
}
```
