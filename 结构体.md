# 结构体 (Structs)

结构体是一种用户自定义的数据类型，用于组合多个不同类型的字段。与类不同，结构体主要用于存储数据，通常包含公共字段，并且不包含方法（尽管在某些实现中可能支持）。

## 定义结构体

使用 `struct` 关键字定义结构体。结构体成员必须显式指定类型。

```korelin
struct Point {
    int x;
    int y;
};
```

## 内联声明变量

可以在定义结构体的同时声明结构体变量。

```korelin
struct Rect {
    int width;
    int height;
} rect1;
```

## 实例化

可以像 C 语言一样直接声明结构体变量。

```korelin
struct Point p; 
p.x = 10;
p.y = 20;
```

## 结构体数组

可以声明结构体类型的数组。

### 声明数组
```korelin
struct Point points[10]; // 声明一个包含 10 个 Point 的数组
points[0].x = 100;
```

### 内联数组声明
在定义结构体时直接声明结构体数组变量。

```korelin
struct Node {
    int id;
} nodes[100]; // 声明一个包含 100 个 Node 的数组变量 nodes
```

## 结构体 vs 类

*   **结构体 (Struct)**: 侧重于数据聚合，默认成员可能是公开的（取决于具体实现，但在本语言中通常作为数据容器）。
*   **类 (Class)**: 侧重于封装和行为（方法），支持访问控制（public/private/protected）和继承。
