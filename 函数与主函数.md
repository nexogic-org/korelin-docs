# 函数与主函数

## 函数定义

Korelin 语言使用显式的类型声明来定义函数，不再使用 `func` 关键字。函数定义的语法如下：

```korelin
返回值类型 函数名(参数列表) {
    // 函数体
}
```

### 基本示例

```korelin
// 定义一个返回整数的函数
int add(int a, int b) {
    return a + b;
}

// 定义一个无返回值的函数
void sayHello(string name) {
    print("Hello, " + name);
}
```

### 泛型函数

支持在函数名后添加泛型参数，用于定义通用算法：

```korelin
// 泛型函数示例
T identity<T>(T value) {
    return value;
}

// 调用泛型函数
int x = identity<int>(10);
```

## 主函数 (main)

每个 Korelin 项目必须且只能包含一个 `main` 函数。这是程序的入口点。

### 规则

1.  **唯一性**：项目中只能定义一个 `main` 函数。如果在不同文件中定义了多个 `main` 函数，编译器会报错 `NameDefineError: 主函数 'main' 重复定义`。
2.  **格式**：
    ```korelin
    返回值类型 main(参数列表) {
        // 程序逻辑
    }
    ```
3.  **返回值**：`main` 函数的返回值类型可以是任意合法的类型（如 `int`, `void`, `string` 等）。整个虚拟机的返回值通常由 `main` 函数的返回值决定。
4.  **参数**：`main` 函数可以接受参数，通常用于处理命令行输入。

### 示例

```korelin
// 标准入口
int main() {
    print("Korelin Program Running...");
    return 0;
}
```

## 递归调用

函数支持递归调用：

```korelin
int fib(int n) {
    if (n <= 1) return n;
    return fib(n - 1) + fib(n - 2);
}
```
