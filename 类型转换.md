# 类型转换 (Type Conversion)

Korelin 提供了多种方式在不同数据类型之间进行转换。

## 1. 显式类型转换 (Explicit Casting)

### 1.1 使用 `os` 模块函数
`os` 模块提供了一组内置函数用于基本类型的转换：

- `os.int(value)`: 将值转换为整数。
  - 字符串: 解析整数 (e.g. "123" -> 123).
  - 浮点数: 截断小数部分 (e.g. 3.14 -> 3).
  - 布尔值: true -> 1, false -> 0.
  
- `os.float(value)`: 将值转换为浮点数。
  - 字符串: 解析浮点数 (e.g. "3.14" -> 3.14).
  - 整数: 转换为浮点 (e.g. 3 -> 3.0).

- `os.str(value)`: 将任意值转换为字符串表示。

### 1.2 示例
```korelin
import os;

var s = "123";
var i = os.int(s); // 123
var f = os.float(s); // 123.0

var n = 456;
var s2 = os.str(n); // "456"
```

## 2. 隐式类型转换 (Implicit Casting)

在某些运算中，Korelin 会自动进行类型提升：

- **算术运算**: `int` + `float` -> `float`.
- **字符串拼接**: `string` + `any` -> `string` (调用 `toString`).

## 3. 类型检查 (Type Checking)

可以使用 `typeof(value)` 获取值的类型名称（字符串）。
